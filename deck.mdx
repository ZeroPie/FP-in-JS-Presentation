Was ist Functional Programming?

```md title='Functional Programming'
The process of building software by composing pure functions.
avoiding shared state, mutable data, and side-effects 
```

```js title='Side Effects'
  A side effect is any application state change that is 
  observable outside the called function other than its return value. 
```

```js title='Pure Function'
  
  Given the same input, will always return the same output.
  
  Produces no side effects.

```
